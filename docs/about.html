About

RBoom is a fork of the popular Doom source port PRBoom+. The name "RBoom" is short for "retarded boom"
( although, sorry if there are some unintended insinuations of that name :( )
This version of the port incorporates numerous additional features and options that are unnecessary.
While prboom+ is a good and worthy source port, rboom reverses that and addes uncompatibility options and makes your gameplay even worse.
If you dont like it, too bad. Go use GZDoom, PRBoom+ or something else.
